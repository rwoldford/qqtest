# Package index

## qqtest functionality

main functions for quantile quantile testing

- [`qqtest()`](https://rwoldford.github.io/qqtest/reference/qqtest.md) :

  `qqtest` A self-calibrated quantile-quantile plot for assessing
  distributional shape.

- [`revealLocation()`](https://rwoldford.github.io/qqtest/reference/revealLocation.md)
  :

  `revealLocation` Revealing locations encoded as a string calculation.

- [`hideLocation()`](https://rwoldford.github.io/qqtest/reference/hideLocation.md)
  :

  `hideLocation` Obfuscating the true location of the real data.

## K distribution

Density, distribution function, quantile function and random generation
for the K distribution

- [`dkay()`](https://rwoldford.github.io/qqtest/reference/dkay.md) :

  `dkay` The density function of the K distribution

- [`pkay()`](https://rwoldford.github.io/qqtest/reference/pkay.md) :

  `pkay` The cumulative distribution function K distribution

- [`qkay()`](https://rwoldford.github.io/qqtest/reference/qkay.md) :

  `qkay` The K distribution quantile function

- [`rkay()`](https://rwoldford.github.io/qqtest/reference/rkay.md) :

  `rkay` The K distribution - generating pseudo-random values

## Data

A collection of historical data sets used to illustrate quantile
quantile plots.

- [`bacteria`](https://rwoldford.github.io/qqtest/reference/bacteria.md)
  : Bacteria from Delaware River water entering the Torresdale Filter of
  the Philadelphia water supply 1913.
- [`penicillin`](https://rwoldford.github.io/qqtest/reference/penicillin.md)
  : 31 contrast sums from a 32 run 2^(5-0) factorial experiment on
  penicillin production.
- [`primer`](https://rwoldford.github.io/qqtest/reference/primer.md) :
  Automobile primer paint thickness quality control measurements.
- [`pullstrength`](https://rwoldford.github.io/qqtest/reference/pullstrength.md)
  : Strength of pull for 519 males aged 23-26.
- [`sittingHeights`](https://rwoldford.github.io/qqtest/reference/sittingHeights.md)
  : Sitting height in inches of female adults (aged 23-50).
- [`stacklossDistances`](https://rwoldford.github.io/qqtest/reference/stacklossDistances.md)
  : Mahalanobis squared distances of Brownlee's stack loss plant
  operation data based only on the explanatory variates (air flow, water
  temperature, and acid concentration).
- [`WachusettReservoir`](https://rwoldford.github.io/qqtest/reference/WachusettReservoir.md)
  : Storage, in millions of gallons daily per square mile of net land
  area, at the Wachusett Reservoir in Massacusetts - storage computed
  for each of several rates of draft (draft being a determined
  maintainable flow in 1,000s of gallons per square mile daily).
